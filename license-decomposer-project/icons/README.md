
# JFrog 图标使用说明

## 📥 如何添加JFrog图标:

1. 将JFrog图标保存为以下任一文件名:
   - `jfrog_icon.png` (推荐)
   - `license_decomposer.png`
   - `app_icon.png`

2. 将图标文件放入 `icons/` 目录

3. 重启应用即可看到新图标

## 📐 推荐的图标规格:
- 格式: PNG (推荐) 或 ICO
- 尺寸: 64x64 到 256x256 像素
- 背景: 透明背景最佳

## 🔄 图标优先级:
1. jfrog_icon.png (最高优先级)
2. license_decomposer.png
3. app_icon.png
4. icon.png
5. 自动生成的默认图标

## 💡 提示:
- macOS Dock图标最佳尺寸: 512x512 或 1024x1024
- Windows ICO文件支持多尺寸
- 状态栏图标推荐: 16x16 或 22x22
